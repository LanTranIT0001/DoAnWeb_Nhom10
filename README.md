# DoAnWeb_Nhom10
# Giới thiệu

Dự án mô phỏng Website Pinterest được thực hiện trong khuôn khổ bài đồ án cuối kỳ học phần Phát triển Ứng dụng Web.  
Dự án tập trung xây dựng giao diện và các chức năng cơ bản của Pinterest như hiển thị hình ảnh dạng lưới, lưu ảnh và quản lý bộ sưu tập.

# Chức năng chính
- Hiển thị danh sách hình ảnh (Pins)
- Bố cục dạng lưới tương tự Pinterest
- Xem chi tiết hình ảnh
- Lưu và quản lý pin (mô phỏng)
- Giao diện responsive

# Công nghệ sử dụng
- HTML
- CSS
- JavaScript
- PHP
- SQL
(Có thể mở rộng thêm tùy theo yêu cầu môn học)
